const std::string &  sName,
bool  bManaged,
bool  bDynamic 
)

Registriert eine Audiosignalquelle.

Diese Methode fügt dem Manager eine bestehende Audiosignalquelle hinzu. Hierbei wird der Audiosignalquelle auf Basis ihres Mnemonics eine ID zugeteilt. Diese Methode u.A. kann benutzt werden, um externe Signalquellen anzubinden.

bManaged = true -> Manager verwirft die Datenquelle wenn er freigegeben wird bDynamic = true -> Quelle kann zur Laufzeit aus dem Manager entfernt werden

Allgemeine AddMethode. Damit können auch eigene Typen (Unterklassen von IVAAudioSignalSource) registriert werden

  • IDs (Zahlen) werden immer erhöht, auch wenn Zahl davor freigegeben wurde
Returns
ID der hinzugefügten Quelle
