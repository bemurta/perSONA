Detailed Description

Klasse zur Bewegungsschätzung.

Diese Klasse implementiert die Grundfunktionalität eines Bewegungsschätzers durch Speichern vorangegangener Positionen. Bewegung wird aus Gewichtung der Historie durch Dreiecksfunktion erreicht.

Note
ACHTUNG: Das Model ist nicht thread-safe!
