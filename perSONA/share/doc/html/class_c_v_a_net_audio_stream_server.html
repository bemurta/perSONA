Detailed Description

Stream server framework for providing samples for a network stream.

This class helps to set up a network audio streaming server to be connected with a VA network stream. Process() method needs to be implemented by user and is called by the network server implementation occasionally if more samples are requested. Timing can be observed with the stream status, i.e. a time stamp and sample count. The timing is most probably not as acurate as an audio stream that is driven by a sound card, because network transmission deviates stream process call timings.
