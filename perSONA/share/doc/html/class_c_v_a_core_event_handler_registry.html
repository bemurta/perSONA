Detailed Description

Diese Klasse regelt die Registrierung von Empfängern für Core-Ereignisse. Handler können sich an-/abmelden und es wird eine Funktion für das Senden eines Ereignissen an alle registrierten Handler bereitgestellt. Alle Methoden der Klasse sind thread-safe.
