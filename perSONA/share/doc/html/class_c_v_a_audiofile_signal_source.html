IVAInterface * CVAAudiofileSignalSource::GetAssociatedCore ( ) const
virtual

Associated core getter.

Returns the pointer to the associated core (IVAInterface) that is registered with the sound source. Will return NULL if the source has not been associated to a core yet. This method is used by the core to assuure that the sound source object has not been associated to another core (not allowed).

Returns
Pointer to the associated IVAInterface, NULL if non present

Stream block getter.

Returns the data pointer to the next audio block of the sound source.

Parameters
[in]oStreamStateInformation on stream state
Returns
Pointer to the audio data array containing data for next block (NULL if no data available)
Note
This method is called by the core to fetch data. If the core receives a NULL pointer it will be interpreted as silence.
