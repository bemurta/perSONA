Detailed Description

Associative array container.

Associative array container, mapping a unique collection of string keys to values. A variety of key values is supported, for instance primitive types (bool, int, double, string). Nesting is also supported by assigning keys another associative array. Keys names are case-insensitive. The class features simple and effective use by defining data access and cast operators. It can be efficiently serialized/deserialized, making it a suitable container for message mechanisms.
