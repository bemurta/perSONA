Detailed Description

Interface class for audio signal sources.

This purely abstract class defines an interface for data source objects that provide audio-streams. It is intended to acoustically represent sound emitting objects in a virtual scene.

A data source is always monaural (single channel) and can only be connected to a single instance of a IVAInterface during runtime.
