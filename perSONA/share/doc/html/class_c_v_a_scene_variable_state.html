Detailed Description

Klassentemplate für Szenevariablen.

Jede Variable hat einen Lebenszyklus. Dieser beginnt mit ihrer Erzeugung und Einfügung in einen Konfigurationsraum. Er endet falls das übergeordnete Objekt (z.B. Schallquelle) aus der Szene gelöscht wird. Allerdings bleibt die Variable trotzdem erhalten, hat aber dann den Zustand 'dead'.
