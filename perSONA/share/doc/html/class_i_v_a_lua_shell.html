Detailed Description

Diese abstrakte Klasse definiert die Schnittstelle für eine Shell welche einen Lua-Interpreter beherbergt und es erlaubt einen VACore mittels Lua anzusprechen und zu steuern.

Konkrete Instanzen der Klasse müssen mit der Factory method erzeugt werden.
