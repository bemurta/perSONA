Getting started

If you are a C++ developer and want to investigate the interface and integrate VA into your application, the first contact point should be the VABase interface represented by the IVAInterface class. If you are planning to create a binding for any other programming and/or scripting language, also have a look at VANet and the IVANetClient class. If you want to host a VA server or want to extend the core of VA by new rendering and reproduction modules, see VACore.
