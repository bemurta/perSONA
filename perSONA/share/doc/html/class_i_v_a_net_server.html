Detailed Description

VA server interface.

This class defines an interface for a TCP/IP network server with a managed VA core.

Use the factory method Create() to instantiate a server, then set a VA core interface instance using SetCoreInstance().

Calls from a network client will be transmitted over TCP/IP transparently, and will be forwarded to the actual core.

The server network functionality allows multiple client connections, however updates are synchronized making them potentially less efficient.
