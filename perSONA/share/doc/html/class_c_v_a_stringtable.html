Detailed Description

Diese Klasse realisiert einen einfachen Stringtable, welcher als assoziativer Container agiert, welcher IDs Zeichenketten zuweist. Er wird u.A. benutzt um Szeneobjekten die Namen zuzuweisen. Die Klasse ist nicht thread-safe.
