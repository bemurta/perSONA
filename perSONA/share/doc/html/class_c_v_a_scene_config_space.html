Detailed Description

Konfigurationsraum.

Ein Konfigurationsraum versioniert eine Menge von Variablen. Er verwaltet einen Versionszähler, welchen in ihm enthaltenen Variablen verwenden. Jede Änderung einer einzelnen Variablen bzw. die synchronisierte Änderung mehrerer Variablen führt zu einer neuen Versionsnummer. Versionsnummer sind stets für die Gesamtheit aller Variablen definiert und nicht für einzelne.
