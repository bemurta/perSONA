bool CVAAudiostreamState::bSyncMod

Flag that indicates, that a synchronized sequence of atomic commands is executed on elements of the stream. If this flag is true the source shall handle parameter changes and buffer them internally, but first apply them for the stream, if the flag is set to false again. Example: Synchronized start/stop of file sources
