Detailed Description

Klasse für Bewegungszustände (Zeitmarke, Position und Geschwindigkeit, Orientierung)

Diese Klasse bescheibt einen dynamischen (versionierten) Bewegungszustand eines Objektes. Sie beinhaltet die Position und Geschwindigkeit sowie die Orientierung des Objektes zu einem gewissen Zeitpunkt.
