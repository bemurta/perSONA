Detailed Description

Audio reproduction module interface.

Diese rein abstrakte Klasse definiert die Schnittstelle für die Wiedergabe binauraler Signale (z.B. Kopfhörer und virtueller Kopfhörer).

Wiedergabe-Module bekommen als Eingabe eine Stereo-Audiosignalquelle zugewiesen. The audio reproduction modules take a rendering stream as input and reproduce the sound field (at least at the eardrums) for a real world listener. They can react upon scene changes, i.e. if the user moves in real world coordinates (in the lab).
