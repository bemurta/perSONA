Detailed Description

Central manager of the auditory scene.

This class is the central manager of the auditory scene. It stores any auditory object in pool-based lists for rapid creation/deletion using individual reference counter. The class provides access to the head state of the scene as well as to any other scene in the past using IDs. It also provides mechanism to derive a scene from a base scene.
